[![License:Apache2.0](https://img.shields.io/badge/License-Apache2.0-yellow.svg)](https://opensource.org/licenses/Apache2.0)

## Install

```shell
git clone git@github.com:makeryangcom/DesktopTemplate.git
```

## Development

```shell
yarn && yarn dev
```

## Build

```shell
yarn build
```

## Pack

```shell
yarn win64
```

```shell
yarn mac
```
