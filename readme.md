# DesktopTemplate

⚡ Vue3 + TypeScript + Vite Desktop Template ⚡

## Install

```shell
git clone git@github.com:makeryangcom/DesktopTemplate.git
```

## Development

```shell
yarn
```

```shell
yarn dev
```

## Build

```shell
yarn build
```

## Pack

```shell
yarn win64
```

```shell
yarn mac
```
